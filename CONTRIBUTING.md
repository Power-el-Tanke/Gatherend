# Contributions Policy

As of right now, I accept contributions but they must be simple and clear, if you'd like to add a whole feature or something like that consult me first in support@gatherend.com.
This project is not open-source, is source-available, in 4 years since its release a version becomes open source, so take that into consideration if you want to contribute. I don't force anyone to contribute, I'm okay working by myself in this project but as I said, if you'd like to add a feature or something I'm open to it but do it step by step in every PR so I don't have to review 500 new lines of a new feature, thanks :D.
