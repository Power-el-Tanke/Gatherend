# Security Policy

## Supported Versions

I only provide security patches for the latest major version of Gatherend.

## Bug Bounty Program

I don't currently operate a bug bounty program.
I do not offer monetary rewards for vulnerability reports.
I appreciate responsible disclosure and will acknowledge contributors in my release notes
(with permission), but submitting a report does not guarantee or imply any monetary compensation.

## Reporting a Vulnerability

If you discover a vulnerability, please report it privately.

Email: security@gatherend.com

Please include:

- Description of the issue
- Steps to reproduce
- Impact assessment

Do NOT open public issues for security vulnerabilities.

## Response Time

I aim to acknowledge reports within 48 hours and provide a fix as soon as possible.
